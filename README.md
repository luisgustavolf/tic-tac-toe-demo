# Tic-tac-toe Demo

is a simple demo project, that implements a tic-tac-toe engine trough a React Hook

## Using this project

### Small issues

- Project may contain some minor issues

### Docker way

```bash
# build once
docker-compose build

# run it
docker-compose up
```

### Classic way

#### Prerequisites 
- node 12.x (or higher)
- yarn 1.22 (or higher)

#### Installation

```bash
yarn
```

#### Tests

```bash
yarn test
```

#### Running the project (dev purposes)

```bash
yarn run
```

#### Building the project

```bash
yarn build
```
